# Roadmap Post-Lanzamiento

Bloques aparcados temporalmente para desarrollar una vez el programa base este lanzado y el bloque fiscal VERIFACTU/SII quede estable.

## Modulos verticales a incorporar

1. Transporte terrestre intermodal.
2. Almacenaje avanzado y distribucion multiproceso.
3. Talleres y reparaciones con flujos ampliados.
4. Transitarios maritimo, aereo y terrestre.
5. Agentes de aduanas y deposito aduanero.
6. Transporte terrestre de contenedores import/export.
7. Comunicaciones con plataformas externas:
   - VAL-PORT
   - PORTIC
   - INTTRA
8. Enlaces contables avanzados.

## Criterio de prioridad acordado

Antes de estos modulos, el foco principal del producto queda en:

1. VERIFACTU
2. SII
3. Cierre fiscal y documental de facturacion
4. Salida estable a produccion

## Nota

Estos modulos se plantean como capas sobre el nucleo actual de:
- trafico
- pedidos
- facturacion
- portal cliente/proveedor
- app chofer
- almacen
- taller
